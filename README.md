# elf-loader
A (mostly incorrect) ELF loader

This project is developed following an amazing [series of articles](https://fasterthanli.me/series/making-our-own-executable-packer), but actual specifications can be found on [the linux foundation website](https://refspecs.linuxfoundation.org/). [The IRELATIVE relocation](https://stackoverflow.com/questions/17404672/what-does-r-x86-64-irelativ-mean) could also be useful.
